# Portfólio UMC
- https://oliveira-iago-temp.github.io/umc-portifolio/

Projeto de portfólio web desenvolvido para atividade acadêmica da UMC, com estrutura de landing page e foco em apresentação profissional.

## Sobre o projeto

O site foi construído com layout responsivo, seções semânticas e animações de entrada conforme a rolagem da página (scroll reveal), atendendo aos requisitos da atividade.

## Stack utilizada

- HTML5 (estrutura semântica)
- CSS3 (layout responsivo, estilos e animações)
- JavaScript (interatividade leve / animações ao rolar)

## Estrutura do projeto

- `index.html`
- `styles.css`
- `script.js`
